===============================================================================
Real-time production observability is achieved using a lightweight, structured 
Local Telemetry Engine integrated directly into the application pipeline.

  - Logging Paradigm: Structured Append-Only File Stream (telemetry_history.jsonl)
  - Telemetry Field Tracking:
    * timestamp (ISO 8601 string sequence format)
    * model_variant (Qwen-0.5B vs. gpt-4o-mini execution tier)
    * turn_count (Sequential integer transaction index tracking session depth)
    * latency_ms (Precise execution delta clock timing for Time-To-First-Token)
    * tokens_processed (Input prompt and generated response lengths)
    * guardrail_status (Pass flag vs. Early-exit refusal interception tags)
